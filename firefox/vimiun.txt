# Keymaping
map down scrollDown
map up scrollUp
map shiftLeft goBack
map shiftRight goForward
unmap n
map n LinkHints.activateModeToOpenInNewTab
map cw removeTab
unmap u
map u restoreTab
map <s-down> nextTab
map <s-up> previousTab
